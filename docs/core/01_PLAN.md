# 01_PLAN.md - 프로젝트 기획서

## 1. 프로젝트 개요
- **프로젝트 명**: AZERC Portfolio
- **한 줄 컨셉**: "The Nexus of AI, Blockchain, and 3D" - 혁신적인 기술의 교차점을 시연하는 프리미엄 포트폴리오
- **목표**: AI(LLM), Blockchain(Web3), 3D Graphics(WebGL) 기술력을 시각적으로 압도하며 전달하고, Full-Stack 개발 역량을 증명함.

## 2. 브랜딩 및 디자인 전략
- **핵심 키워드**: Futuristic, Premium, Integration, Interactive, Robust.
- **디자인 테마**: 
    - **Glassmorphism**: 투명도와 블러 효과를 활용한 현대적이고 깊이감 있는 UI.
    - **Dark Immersion**: 깊은 다크 모드를 기반으로 하여 3D 요소와 네온 컬러 가독성 극대화.
    - **Micro-Animations**: Framer Motion을 활용한 부드러운 전환과 피드백.
- **메인 컬러**: 
    - Primary: #00F2FF (Electric Cyan)
    - Secondary: #7000FF (Hyper Purple)
    - Accents: #FF00E5 (Cyber Pink)

## 3. 핵심 기능 (Key Features)

### 3.1. Interactive Project Showcase
- **3D Card Layout**: 각 프로젝트를 Three.js 또는 Framer Motion 3D로 시각화한 카드.
- **Tech Stack Chips**: 프로젝트별 사용 기술(AI, Blockchain 등)을 태그 형태로 표시.
- **Live Preview & Repo**: 실제 서비스 링크와 GitHub 저장소 연결.

### 3.2. Technical Deep-Dive
- **Architecture Diagrams**: 각 프로젝트의 시스템 아키텍처를 Mermaid 등을 이용해 전문적으로 설명.
- **Problem & Solution**: 개발 중 직면했던 기술적 난제와 해결 과정을 상세 기록.

### 3.3. AI Concierge (Option)
- **Portfolio Guide**: 사용자의 질문에 맞춰 관련 프로젝트를 추천하거나 제작 기술을 설명하는 챗봇 인터페이스.

## 4. 기술 스택 (Tech Stack)
- **Frontend**: Next.js 15 (App Router), Tailwind CSS v4, Framer Motion, Three.js (React Three Fiber).
- **Backend/DB**: React Router v7 (Server Side logic), Better Auth, Drizzle ORM, SQLite.
- **Infrastructure**: Vercel.

## 5. 단계별 로드맵 (Roadmap)
- **Phase 1: Foundation**: 기획 문서 확정 및 기본 UI 레이아웃 구축.
- **Phase 2: Project Integration**: 주요 4대 프로젝트(BondBase, Three.js Lab 등) 데이터 구성.
- **Phase 3: Interactive Polish**: 3D 요소 추가 및 애니메이션 최적화.
- **Phase 4: Launch**: Vercel 배포 및 최종 성능 점검.
